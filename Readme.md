# 💻 Shell Scripts Description

## 🟡 Script 2 — Package Inspector

## 🔵 Script 3 — Disk & Permission Auditor
Shows permissions and size of important directories and checks Git configuration file.

---

## 🔴 Script 4 — Log File Analyzer
Counts occurrences of a keyword in a log file and shows recent matching lines.

## 🟣 Script 5 — Manifesto Generator
